🌐 Agentsafe Flow

How a user connects & how an agent spends safely on Base

1

Connect Wallet

User opens agentsafe.app → clicks "Connect Wallet" → MetaMask/WalletConnect pops up.

2

Sign Message (No Gas)

Server sends a nonce. User signs personal_sign. Free — no transaction, no gas fee.

3

Server Verifies → JWT Session

Signature verified against wallet address. Server issues 24h JWT token. User is now authenticated.

4

Generate API Key for Agent

Dashboard creates an API key scoped to this wallet. Developer pastes key into their agent code.

5

Agent Runs → Hits Paid API

Agent makes a request. API returns HTTP 402 (payment required). SDK intercepts.

6

Safety Gate Check

Server runs all 8 guards: Budget ✓ Trust ✓ Anomaly ✓ KillSwitch ✓
If all pass → APPROVED + signed payment header.

7

x402 Settlement on Base

EIP-3009 gasless USDC transfer. Agent gets access. Every spend logged to Merkle audit chain.

Base Mainnet (8453) · USDC (6 decimals) · x402 protocol · Wallet auth
